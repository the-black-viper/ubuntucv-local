# ubuntucv-local
Build ubuntu 18.04 with OpenCV 4 Locally.

### Install dependencies in order
- build-essential
- cmake
- git
- libgtk2.0-dev
- libavcodec-dev
- libavformat-dev
- libswscale-dev
- python-dev
- python3-dev
- python-numpy
- python3-numpy
- libtbb2
- libtbb-dev
- libjpeg-dev
- libtiff-dev
- unzip
- wget
- checkinstall (optional)

All other OpenCV dependencies can be installed subsequently.
